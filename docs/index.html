Recursos para uma Experiência Premium

Construído do zero para suprir as limitações do editor original, aliando estabilidade com tecnologias modernas de desenvolvimento.

Editor com CodeMirror 6

Destaque de sintaxe inteligente para comandos do Visualg, numeração de linhas, autocomplete de comandos e variáveis declaradas dinamicamente.

Depuração por Etapa

Monitore variáveis, tipos, saída acumulada e expressões de inspeção. Avance passo a passo, volte pelos passos já visitados ou continue até o próximo ponto de parada, com Worker e ponte assíncrona para entradas.

Fluxogramas Automáticos

Visualização gráfica do algoritmo gerada instantaneamente a partir da Árvore de Sintaxe Abstrata (AST) do interpretador. Com exportação para SVG e PNG.

Múltiplos Temas Visual

Chega de luz branca excessiva. Use temas como Dracula, Dark e Visual Assist Dark, além de ajustar fontes e guias de indentação.

Editor Multi-Abas

Mantenha várias abas abertas simultaneamente, com salvamento de estado automático, indicador de alterações pendentes e atalhos rápidos.

Interpretador Próprio

Interpretador em Node.js modular, compatível com a linguagem original, com suporte a subprogramas, funções nativas, vetores, matrizes e Worker para execução/depuração com fallback técnico.

Análise de Complexidade

Monitore a Complexidade Ciclomática e a estimativa de Big O em tempo real diretamente na barra de status, estimulando boas práticas de escrita e eficiência.

Atualizações pelo GitHub

Releases públicas com instalador Windows, pacotes Linux, checksums SHA-256 e metadados para auto-update diretamente pela tela Sobre do aplicativo.

Por que usar o VisuAlg Studio?

Reconstruímos a experiência do Visualg adaptada para o que os desenvolvedores esperam das ferramentas de hoje.

  • Interface Responsiva e Customizável

    Arraste e redimensione os painéis de código, console e depuração como achar melhor para a sua tela.

  • Preenchimento Dinâmico de Entradas

    Insira entradas leia(...) múltiplas antes da execução ou responda via modais interativos no momento exato. O Worker pede a entrada para a UI sem trocar a experiência visual.

  • Conformidade com a Sintaxe Original

    Suporta vetores multidimensionais, ranges em caso, passagem por referência com var, recursão e funções nativas matemáticas e de manipulação de string.

  • Métricas de Complexidade Pedagógicas

    Apoio ao aprendizado de ciência da computação com cálculo de caminhos lógicos (complexidade ciclomática) e estimativas Big O (com suporte a anotações personalizadas).

  • Distribuição e Atualização Simples

    Builds para Windows e Linux são publicados no GitHub Releases com checksums e metadados usados pelo auto-updater integrado ao app.

Perguntas Frequentes (FAQ)

Tire suas dúvidas sobre o VisuAlg Studio e saiba como ele ajuda no aprendizado de algoritmos.

O que é Portugol?

Portugol é uma pseudolinguagem estruturada em português usada para ensinar lógica de programação e algoritmos. Ela permite que estudantes iniciantes foquem na estrutura e na lógica do código, sem as barreiras e complexidades de sintaxe das linguagens de programação comerciais em inglês.

Qual a diferença entre Visualg e VisuAlg Studio?

O Visualg original é uma ferramenta clássica desenvolvida em Delphi e restrita ao ecossistema Windows legado. Já o VisuAlg Studio é uma reconstrução desktop moderna em código aberto baseada em Electron e Node.js. Ele oferece uma interface Dracula muito mais rápida, abas para arquivos, realce e autocompletar inteligente com CodeMirror 6, geração dinâmica de fluxogramas e execução segura através de Web Workers, mantendo compatibilidade com o Portugol tradicional.

Como instalar o VisuAlg Studio no Linux?

No Linux, você pode baixar o arquivo .AppImage ou o pacote .deb na página de Releases. Para a versão portátil .AppImage, basta dar permissão de execução: 

chmod +x VisuAlg-Studio-X.Y.Z.AppImage
./VisuAlg-Studio-X.Y.Z.AppImage

Ou instalar o pacote Debian/Ubuntu usando o gerenciador de pacotes: 

sudo dpkg -i visualg-studio_X.Y.Z_amd64.deb
